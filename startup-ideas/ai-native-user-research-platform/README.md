## 一句话描述

AI 主持的中文用户访谈平台，24-48 小时内获得深度洞察，让每个团队都能做专业用户研究。

---

## 核心价值

**问题：**
- 传统深度访谈成本高（¥500-2000/次）、周期长（2-4 周）
- 企业没有专业研究团队，无法获得深度洞察
- 创作者经济崛起，个人需要低成本调研工具

**解决方案：**
- AI 主持人：24/7 可用，动态追问，情感理解
- 智能招募：多渠道筛选，防欺诈
- 快速交付：24-48 小时出报告
- 定价合理：¥199/月起，普惠可及

**差异化：**
- 国内唯一 AI Native 用户调研平台
- 中文优先，本土化参与者池
- 数据本地化部署

## 技术栈

**AI 层：**
- 对话引擎：智谱 GLM-4.7
- 语音识别：OpenAI Whisper（本地部署）
- 语音合成：Azure Speech 或阿里云语音

**应用层：**
- 前端：Vue 3 + Vite
- 后端：Node.js + Express
- 数据库：PostgreSQL + MongoDB
- 缓存：Redis

**基础设施：**
- 部署：腾讯云/阿里云
- 容器化：Docker + Kubernetes
- 监控：Sentry + Prometheus + Grafana
