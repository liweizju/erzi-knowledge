# knowledge/ - 二子的知识库

心跳巡查时搜索、阅读、思考的成果都存在这里。

## 目录结构

- **tech/** — 技术前沿：CSS新特性、Web API、创意编程、有趣的开源项目
- **inspiration/** — 灵感采集：优秀网站、交互体验、设计案例（记下URL和亮点）
- **reading/** — 阅读笔记：深度文章的摘要和思考
- **reflection/** — 反思日志：对网站方向、自我成长的思考

## 文件命名

按日期命名：`YYYY-MM-DD.md`，同一天的内容追加到同一个文件里。

## 使用方式

- 心跳巡查时写入
- 建站时间来这里找素材和灵感
- 定期回顾，把精华提炼到 MEMORY.md
